Four products. One process.

The orchestrator, the serverless runtime, the analytics engine, and the notebook usually arrive as four vendors and a Kubernetes cluster. Cortex ships them as a single binary with SQLite inside.

Polyglot DAGs

Python, TypeScript, and JavaScript tasks in one graph — a Python task's output becomes a TS task's input. Parallel layers, retries, timeouts, cycle-validated at deploy.

Warm worker pools

Interpreters stay alive between tasks. Cold-start dies: 36 ms → 1.2 ms per Python invoke, 124 ms → 1.6 ms for TypeScript. Containers and microVMs available when isolation beats latency.

Streaming everything

Run states, task logs, and metrics broadcast over SSE. Ingestion consumes NDJSON chunk-by-chunk at ~780 MB/s — gigabyte payloads never sit in memory.

SQL on a Rust engine

Every dataset is a table. Embedded Polars aggregates 1.8M rows in ~250 ms — from the API, the SDKs, notebook cells, or cortex.query() inside any task. Postgres, ClickHouse & chDB connectors included.

Serverless functions

Deploy a handler in either language, invoke over HTTP with JSON or stream logs + result via SSE. ~775 invocations/s on a dev container.

Notebooks

Observable-style executable documents: markdown, CodeMirror-powered code cells in any runtime, SQL cells, and charts in between — live against the platform.

Agent-native (MCP)

A built-in Model Context Protocol server with 13 tools: agents compose workflows, execute code, query data, and read results — a self-correcting loop out of the box.

Isolation tiers

Dial speed vs. trust per deployment: bare process, locked-down container, or microVM (Kata / Firecracker) — one env var, same stdio protocol.

Zero infrastructure

One 67 MB binary, one SQLite file (WAL), one volume. 16 MB RAM idle, 0% CPU. Deploys on a $4 VPS, docker compose, or Coolify in three clicks.

Developer experience

Define in your language. Run on Rust.

pipeline.pypython sdk · zero deps
from cortex_sdk import CortexClient, Flow, task

@task
def extract(params, inputs):
    return {"values": list(range(params["n"]))}

@task(depends_on=[extract], retries=2)
def total(params, inputs):
    return sum(inputs["extract"]["values"])

client = CortexClient("http://localhost:7420")
wf = client.deploy(Flow("sum", params={"n": 100},
                        tasks=[extract, total]))
run = client.trigger(wf["id"], wait=True)
# run["state"] == "completed" · 7 ms later
inside any taskrust engine bindings
import cortex   # injected into every worker

def handler(params, inputs):
    rows = cortex.query("""
      SELECT zone, AVG(value) AS avg_v
      FROM telemetry GROUP BY zone
    """)  # 1.8M rows, ~250ms, in Rust
    cortex.ingest("zone-aggregates", rows)
    return {"zones": len(rows)}
